Solution to the climate change example
Denis Valle
February 2020

Modified model

I will represent sdev squared for the i-th observation in my model as \(\sigma_i^2\). My new model is givey by:

\[p(\mu,\tau^2|...)\propto [\prod_i N(x_i|\mu,\sigma^2_i\tau^2)]\times N(\mu|m_0,v_0^2)Gamma(\frac{1}{\tau^2}|a,b)\]

Full conditional distributions (FCD)

To develop a Gibbs sampler for this model, I will need to derive the FCDs for each parameter. Here they are:

  • FCD for \(\mu\):

\[p(\mu,|...)\propto [\prod_i N(x_i|\mu,\sigma^2_i\tau^2)]\times N(\mu|m_0,v_0^2)\] \[= N(\mu_1,\tau_1^2)\] where \[\tau_1^2 = [\frac{1}{\tau^2} (\sum_i \frac{1}{\sigma^2_i}) + \frac{1}{v_0^2}]^{-1}\] \[\mu_1=\tau_1^2(\frac{m_0}{v_0^2}+\frac{1}{\tau^2} \sum_i \frac{x_i}{\sigma^2_i})\]

  • FCD for \(\frac{1}{\tau^2}\):

\[p(\frac{1}{\tau^2}|...)\propto [\prod_i N(x_i|\mu,\sigma^2_i\tau^2)]\times Gamma(\frac{1}{\tau^2}|a,b)\] \[=Gamma(\frac{n}{2}+a,b+\sum_i \frac{(x_i-\mu)^2}{2\sigma^2_i})\]

Gibbs sampler functions

I wrote one function to sample \(\mu\) and another function to sample \(\tau^2\). These functions are stored in the R file climate change functions.R and the content of this file is provided below:

sample.mu=function(tau2,sig2,v02,x,m0){
  #calculate variance
  tmp=sum(1/sig2)
  prec=(1/tau2)*tmp+(1/v02)
  var1=1/prec
  
  #calculate mean
  tmp=sum(x/sig2)
  part.mean=(1/tau2)*tmp+(m0/v02)
  mean1=var1*part.mean
  
  rnorm(1,mean=mean1,sd=sqrt(var1))
}
#-------------------------
sample.tau2=function(x,sig2,a,b,mu){
  tmp=((x-mu)^2)/(2*sig2)
  b1=sum(tmp)+b
  a1=a+length(x)/2
  1/rgamma(1,a1,b1)
}

Analysis of simulated data

To see if this Gibbs sampler really works, the first step is to generate some simulated data. I will do this using the following code:

rm(list=ls(all=TRUE))
set.seed(20)
nobs=300

#true parameter values
sig2=runif(nobs,min=0.1,max=2)
tau2=1.5
mu=2

#generate data
x=rnorm(nobs,mean=mu,sd=sqrt(sig2*tau2))
plot(density(x),type='l')

fim=data.frame(x=x,sig2=sig2)

setwd('U:\\uf\\courses\\bayesian course\\example climate change')
write.csv(fim,'fake data.csv',row.names=F)

Here is the code that I use to run the Gibbs sampler.

rm(list=ls(all=TRUE))
set.seed(20)

#import data
setwd('U:\\uf\\courses\\bayesian course\\example climate change')
source('climate change functions.R')
dat=read.csv('fake data.csv',as.is=T)
x=dat$x
sig2=dat$sig2

#initial values
tau2=1
mu=0

#priors
a=b=0.1
m0=0
v02=100

#stuff for Gibbs sampler
ngibbs=1000
store.param=matrix(NA,ngibbs,2)
for (i in 1:ngibbs){
  mu=sample.mu(tau2=tau2,sig2=sig2,v02=v02,x=x,m0=m0)
  tau2=sample.tau2(x=x,sig2=sig2,a=a,b=b,mu=mu)
  store.param[i,]=c(mu,tau2)
}
colnames(store.param)=c('mu','tau2')
plot(store.param[,'mu'],type='l')
abline(h=2,col='red')

plot(store.param[,'tau2'],type='l')
abline(h=1.5,col='red')

Based on the resulting trace-plots, it seems that our algorithm does a good job in retrieving the true parameter values.

Analysis of real data

I analyze the real data using approximately the same code as above, except that I swap the simulated data for the real data. Here is the code that I use:

rm(list=ls(all=TRUE))
set.seed(20)

#import data
setwd('U:\\uf\\courses\\bayesian course\\example climate change')
source('climate change functions.R')
dat=read.csv('climate change data edited.csv',as.is=T)
x=dat$deltaT
sig2=dat$sdev^2

#initial values
tau2=1
mu=0

#priors
a=b=0.1
m0=0
v02=100

#stuff for Gibbs sampler
ngibbs=1000
store.param=matrix(NA,ngibbs,2)
for (i in 1:ngibbs){
  mu=sample.mu(tau2=tau2,sig2=sig2,v02=v02,x=x,m0=m0)
  tau2=sample.tau2(x=x,sig2=sig2,a=a,b=b,mu=mu)
  store.param[i,]=c(mu,tau2)
}
colnames(store.param)=c('mu','tau2')
plot(density(store.param[,'mu']),type='l')
abline(v=mean(store.param[,'mu']),lty=3)
abline(v=mean(x),col='blue') #how different is this from mean of the data?

This plot shows that the posterior mean is substantially different from the mean of the data, suggesting that climate change has been less extreme than we had originally anticipated based on the data alone. This is likely due to the fact that we weigh each observation differently depending on the uncertainty associated with that measurement, given by \(\sigma^2_i\).
