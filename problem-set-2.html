  1. Add a figure to illustrate each distribution. Use three different sets of parameters to illustrate the different shapes these distributions can take. Here is an example for the beta distribution:
set.seed(1)
n=10000
par(mfrow=c(3,1),mar=c(3,3,4,1))
param=matrix(c(30,30,1,10,10,1),3,2,byrow=T)
for (i in 1:3){
  x=rbeta(n,param[i,1],param[i,2])
  seq1=seq(from=0,to=1,by=0.01)
  nomes=paste0('Beta(',param[i,1],',',param[i,2],')')
  hist(x,probability=T,breaks=seq1,main=nomes,
       cex.axis=2,cex.main=3)
  seq1=seq(from=0,to=1,length.out=1000)
  lines(seq1,dbeta(seq1,param[i,1],param[i,2]),col='red',lwd=3)
}
