Implementing the detection limit model
Denis Valle
March 2021

FCD for latent variable \(z_i\)

In our model, we have assumed that:

\[If \; z_i>dl \; then \; y_i=z_i\] \[If \; z_i<dl \; then \; y_i=dl\]

where dl is the detection limit. Therefore, the key difference between this model relative to a regular Gaussian regression is that we have to estimate \(z_i\) whenever \(y_i=dl\).

If we didnt have an observation associated with \(z_i\), then the full conditional distribution (FCD) for \(z_i\) would be simply given by:

\[z_i \sim N(\beta_0 + \beta_1 x_i, \sigma^2)\]

However, if \(y_i = dl\) then the only thing that we know for sure is that \(z_i < dl\) based on our modeling assumptions. Therefore, the FCD for \(z_i\) is given by a truncated normal distribution, where this distribution is truncated from above by dl:

\[z_i|y_i = dl \sim N(\beta_0 + \beta_1 x_i,\sigma^2)I(z_i<dl)\]

FCD for the other parameters

Recall that FCDs assume that we know the value of all the parameters except for the one that we want to sample. In other words, when we sample \(\beta_0\), \(\beta_1\), and \(\sigma^2\), we can assume that we know all \(z_i\)s. As a result, the FCDs for these parameters are the same as the ones used for a standard Gaussian regression model. Here they are:

a - \(\beta_0 \sim N([\frac{n}{\sigma^2} + \frac{1}{10}]^{-1} \frac{1}{\sigma^2} \sum_i (z_i - \beta_1 x_i),[\frac{n}{\sigma^2} + \frac{1}{10}]^{-1})\)

b - \(\beta_1 \sim N([\frac{1}{\sigma^2} (\sum_i x_i^2) + \frac{1}{10}]^{-1} \frac{1}{\sigma^2} \sum_i x_i (z_i -\beta_0),[\frac{1}{\sigma^2} (\sum_i x_i^2) + \frac{1}{10}]^{-1})\)

c - \(\frac{1}{\sigma^2}\sim Gamma(\frac{n+2a}{2},\frac{1}{2}[\sum_i (z_i -\beta_0-\beta_1 x_i)^2] +b)\)

Sampling from a truncated normal distribution

Here is the function that we will need to sample from a truncated normal distribution. For this function, we need to specify the truncation points lo and hi, the mean mu and standard deviation sig of the untruncated normal distribution, and the number of random variables n.

tnorm <- function(n,lo,hi,mu,sig){   #generates truncated normal variates based on cumulative normal distribution
  #normal truncated lo and hi
  
  q1 <- pnorm(lo,mu,sig) #cumulative distribution
  q2 <- pnorm(hi,mu,sig) #cumulative distribution
  
  z <- runif(n,q1,q2)
  z <- qnorm(z,mu,sig)
  z[z == -Inf]  <- lo[z == -Inf]
  z[z == Inf]   <- hi[z == Inf]
  z
}
