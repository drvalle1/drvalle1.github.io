Climate change example

In class, we analyzed the climate change data edited.csv dataset using the following model:

\[p(\mu|...)\propto [\prod_i N(x_i|\mu,\sigma^2)]\times N(\mu|m_0,v_0^2)\] In class, we assumed that all observations have the same amount of measurement error (given by \(\sigma^2\)) and that we knew what \(\sigma^2\) was.

In this exercise, the goal is to create a Gibbs sampler to not only estimate \(\mu\) but also to estimate \(\sigma^2\). As a result, we will need a prior for \(\frac{1}{\sigma^2}\), which we will assume to be a Gamma distribution. Therefore, our full model is given by:

\[p(\mu,\sigma^2|...)\propto [\prod_i N(x_i|\mu,\sigma^2)]\times N(\mu|m_0,v_0^2)Gamma(\frac{1}{\sigma^2}|a,b)\]

Here are the steps to estimate these two parameters:

  1. Determine the FCDs for \(\mu\) and \(\sigma^2\). This has already been done for you and these FCDs are described here.
  2. Create a function in R to sample \(\mu\) from its FCD
  3. Create a function in R to sample \(\sigma^2\) from its FCD
  4. Develop your Gibbs sampler

How can we take into account that each proxy has its own measurement error?

Different proxies clearly have different levels of measurement error. Lets assume that the column sdev indicates how much more variable one proxy is in relation to other proxies.

  • What would be a reasonable model under this assumption?
  • How would we go about creating a Gibbs sampler to estimate the parameters in this model?

For this assignment, you should create a moodel that takes into account the information from the column sdev. Then, you should create a Gibbs sampler to estimate the parameters of this model.
