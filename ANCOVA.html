Linear models ANCOVA
Denis Valle
February 2022

Intro

In ANCOVA, we want to compare two or more groups while at the same time taking into account the effect of a continuous variable. For example, we might want to evaluate the effect of 2 different educational interventions for children in improving spelling while at the same time taking into account age differences. In this example, we would have;

  1. a control group that did not receive any intervention;

  2. a treatment group that received one of the educational interventions; and

  3. another treatment group that received the other educational intervention.

Here is how I would simulate data for this model assuming these 3 groups:

rm(list=ls())
set.seed(1)

#simulation settings
b1=1
sd1=0.5
interc1=1
interc2=1.5
interc3=2

#simulate data
nobs.per.grp=50
age1=runif(nobs.per.grp,min=3,max=9)
y1=rnorm(nobs.per.grp,interc1+b1*age1,sd=sd1)
age2=runif(nobs.per.grp,min=3,max=9)
y2=rnorm(nobs.per.grp,interc2+b1*age2,sd=sd1)
age3=runif(nobs.per.grp,min=3,max=9)
y3=rnorm(nobs.per.grp,interc3+b1*age3,sd=sd1)

#look at data
par(mfrow=c(1,1),mar=c(4,4,1,1))
plot(age1,y1,ylim=range(c(y1,y2,y3)),xlab='Age',ylab='Spelling ability')
lines(c(3,9),interc1+b1*c(3,9))
points(age2,y2,col='blue')
lines(c(3,9),interc2+b1*c(3,9),col='blue')
points(age3,y3,col='red')
lines(c(3,9),interc3+b1*c(3,9),col='red')

This graph suggests a clear effect of age in spelling ability for all groups. Furthermore, we can see that these educational treatments (blue and red lines) increased spelling ability relative to the control group (black line) across all ages but that the 2nd treatment (red line) tended to outperform the 1st treatment (blue line).

Frequentist ANCOVA

Here is how I would fit these data using a standard ANCOVA model:

dat=data.frame(y=c(y1,y2,y3),
               age=c(age1,age2,age3),
               x=c(rep(1,nobs.per.grp),
                   rep(2,nobs.per.grp),
                   rep(3,nobs.per.grp)))
dat$x=as.factor(dat$x)

#fit ANOVA
mod=aov(y~x+age,data=dat)
summary(mod)

ANCOVA as a frequentist linear model

I will rely on the same tricks we used for ANOVA. In other words, I will create 2 binary variables because I have 3 treatments. I will assume the following covariate values for each treatment:

  1. control group: \(x_1=0,x_2=0\)

  2. treatment 1: \(x_1=1,x_2=0\)

  3. treatment 2: \(x_1=0,x_2=1\)

In this example, I chose the control group as the baseline group. According to these covariate values and our linear model equation \(\beta_0 + \beta_1 x_1 +...\), we have the following means for each group:

  1. the mean of the control group is given by \(\beta_0 + \beta_1 \times 0 + \beta_2 \times 0 + \beta_3 \times age=\beta_0 + \beta_3 \times age\)

  2. the mean of treatment 1 is given by \(\beta_0 + \beta_1 \times 1 + \beta_2 \times 0 + \beta_3 \times age=(\beta_0 + \beta_1) + \beta_3 \times age\)

  3. the mean of treatment 2 is given by \(\beta_0 + \beta_1 \times 0 + \beta_2 \times 1 + \beta_3 \times age=(\beta_0 + \beta_2) + \beta_3 \times age\)

Notice that the mean of each group is given by an equation that describes a straight line. Furthermore, all of these lines have the same slope (i.e., they are parallel), given by \(\beta_3\). However, each group has a potentially different intercept, given by \(\beta_0\), \(\beta_0+\beta_1\), and \(\beta_0 + \beta_2\).

I will start by formatting the data to then fit the ANCOVA model as a frequentist linear model:

dat=data.frame(y=c(y1,y2,y3),
               age=c(age1,age2,age3),
               x1=c(rep(0,nobs.per.grp),
                    rep(1,nobs.per.grp),
                    rep(0,nobs.per.grp)),
               x2=c(rep(0,nobs.per.grp),
                   rep(0,nobs.per.grp),
                   rep(1,nobs.per.grp)))

#fit ANCOVA
mod=lm(y~x1+x2+age,data=dat)
summary(mod)

ANCOVA as a Bayesian linear model

Now I will feed the formatted data into our Bayesian linear model:

#create design matrix
xmat=data.matrix(cbind(1,dat[,c('x1','x2','age')]))
y=dat$y
nobs=nrow(dat)
nparam=ncol(xmat)

#fit Bayesian linear model
library(jagsUI)

# MCMC settings 
ni <- 10000 #number of iterations
nt <- 5    #interval to thin 
nb <- 5000  #number of iterations to discard as burn-in
nc <- 3     #number of chains

#set parameters to monitor
params=c("betas","sd")

#run model
setwd('U:\\uf\\courses\\bayesian course\\group activities\\7A Linear models')
dat1=list(y=y,xmat=xmat,nobs=nobs,nparam=nparam)
model2=jags(model.file="linear_model.R",
            parameters.to.save=params,
            data=dat1,n.chains=nc,
            n.burnin=nb,n.iter=ni,n.thin=nt,DIC=TRUE)

The estimated slope for age is close to the true value used in the simulation:

slope.age=model2$sims.list$betas[,4]
plot(slope.age,type='l')

mean(slope.age)
## [1] 1.003648

We can also check if we were able to estimate well the intercepts of each group:

interc=cbind(model2$sims.list$betas[,1],
             model2$sims.list$betas[,1]+model2$sims.list$betas[,2],
             model2$sims.list$betas[,1]+model2$sims.list$betas[,3])
boxplot(interc)

Finally, we can calculate 95% credible intervals for these intercepts to see if they are significantly different between groups:

apply(interc,2,quantile,c(0.025,0.975))
##            [,1]     [,2]     [,3]
## 2.5%  0.7157923 1.079272 1.694654
## 97.5% 1.3411376 1.714123 2.328512

These results suggest that there is a bit of overlap between groups 1 and 2 and groups 2 and 3. However, group 3 is clearly different from group 1.
