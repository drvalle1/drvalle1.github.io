Problem 1

Say that we have the following generative model for the number of fruits produced by a Brazil nut tree:

\[A_i \sim Bernoulli (\pi) \] where \(A_i\) is a latent variable indicating if tree i is an adult or not (\(A_i=1\) and \(A_i=0\), respectively). We assume that the number of fruits \(F_i\) produced by tree i, given that this tree is an adult, is given by

\[F_i | A_i = 1 \sim Poisson (\lambda) \] On the other hand, we assume that if the tree is not an adult, it cannot produce any fruit. In other words,

\[p(F_i=0|A_i=0)=1\]

In this model, we are interested in estimating \(\pi\) and \(\lambda\).

Imagine now that the probability of being an adult tree \(\pi\) is a function of tree diameter \(D_i\) (i.e., larger trees are more likely to be adult trees). Furthermore, suppose that the mean fruit production \(\lambda_i\) depends on the amount of canopy exposure \(C_i\) (i.e., adult trees that receive more sun tend to produce more fruits). In this case, the generative model is given by:

\[A_i \sim Bernoulli(\pi_i = \frac{exp(-0.2+0.05 D_i)}{exp(-0.2+0.05D_i)+1})\] \[F_i |A_i=1 \sim Poisson(\lambda_i=exp⁡(1+2C_i ) )\] \[p(F_i=0|A_i=0)=1\]

Assume diameters range from 5 to 100 cm and that canopy exposure ranges from 0 to 1. Generate 1,000 observations from this generative model and create two scatter plots:

  1. \(C_i\) and \(F_i\); and
  2. \(D_i\) and \(F_i\).

Obs: you can generate diameter and canopy exposure using

  • d1=seq(from=5,to=100,length.out=1000) (for diameter)

  • c1=runif(1000) (for canopy exposure)

Problem 2

Imagine that we are interested in understanding the determinants of fish length \(L_i\) at 100 days of age. Say potential covariates are protein content of food \(P_i\) (continuous variable between 0 and 1) and disease state \(I_i\) (equal to one if diseased and zero otherwise). Because \(L_i\) is a continuous positive response variable, perhaps it is fine to assume a Gamma regression, given by:

\[L_i \sim Gamma(a_i,b_i)\] \[E[L_i]=\frac{a_i}{b_i}=exp(0.3+0.7 P_i - 0.5 I_i)\] \[Var(L_i)=\frac{a_i}{b_i^2}=0.1\]

Generate 1,000 observations from this generative model and create a scatterplot of \(P_i\) and \(L_i\). Color code the observations in this scatter plot based on disease state.

Obs: Generate protein content and disease status using:

  • p1=seq(from=0,to=1,length.out=1000) (protein content)

  • i1=rbinom(1000,size=1,prob=0.5) (for disease state)
